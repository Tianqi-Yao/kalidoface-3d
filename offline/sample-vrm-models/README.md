# 免费 VRM 示例模型

来源:[madjin/vrm-samples](https://github.com/madjin/vrm-samples)(VRoid 官方/社区整理的示例模型仓库)。

- `AvatarSample_A.vrm` / `AvatarSample_B.vrm` / `AvatarSample_C.vrm`
  官方 VRoid Studio 示例模型,几乎所有 VRM 应用(包括 Kalidoface 3D 官网)都用它们做默认演示角色。可自由使用/修改/分发,但版权不放弃,需遵守 [pixiv 官方使用条款](https://vroid.pixiv.help/hc/en-us/articles/4402394424089)。
- `Vivi.vrm` / `Sendagaya_Shino.vrm`
  VRoid Studio 内置模板角色,同一个仓库收录,具体授权条款仓库 README 未逐个标注,个人测试/非商用没问题;如果要公开直播或商用,建议去 [VRoid 官方说明页](https://vroid.pixiv.help/hc/en-us/articles/4402614652569) 确认清楚再用。

## 怎么用

打开 kalidoface-3d(`offline/`),把这个文件夹里任意一个 `.vrm` 文件拖拽到浏览器页面里即可加载,会自动保存到浏览器本地存储,下次打开不用重新拖。
